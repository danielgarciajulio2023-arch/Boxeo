# Boxeo# Boxing Products Store Dashboard

##  Descripción del Problema
La gestión de inventario y ventas de productos deportivos suele ser compleja y dispersa. Este proyecto busca centralizar la administración de productos de boxeo en una aplicación web moderna, ofreciendo un panel visual que permita controlar existencias, pedidos, estadísticas y rendimiento semanal.  
De esta manera, se facilita la toma de decisiones y se mejora la experiencia tanto para administradores como para clientes.

##  Objetivo
Construir una **SPA**con React y Vite que sirva como dashboard para la tienda de productos de boxeo, integrando métricas, tablas y secciones dinámicas con un diseño moderno y escalable.

## Tecnologías Utilizadas
- **React**: Librería para construir interfaces dinámicas.
- **Vite**: Herramienta de desarrollo rápida y moderna.
- **NPM**: Gestor de dependencias.
- **CSS (Flexbox / Grid)**: Para estilos organizados y responsivos.
- **Chart.js (opcional)**: Para visualización de estadísticas.
- **JavaScript (ES6+)**: Lógica de componentes y dinámicas.
